# 画面仕様書

## 1. 画面一覧

| 画面ID | 画面名 | URL | 概要 |
|---|---|---|---|
| SCR-001 | タスク一覧画面 | /tasks | 登録済みタスクの一覧表示と新規作成 |
| SCR-002 | タスク詳細画面 | /tasks/{id} | タスクの詳細表示、ステータス更新、削除 |

## 2. SCR-001 タスク一覧画面

### 画面レイアウト

```
┌─────────────────────────────────────────────┐
│  タスク管理                                    │
├─────────────────────────────────────────────┤
│                                               │
│  [＋ 新規タスク作成]                            │
│                                               │
│  ┌─────────┬──────────┬──────────┬─────┐    │
│  │ タイトル  │ ステータス │ 作成日     │ 操作 │    │
│  ├─────────┼──────────┼──────────┼─────┤    │
│  │ タスクA   │ TODO      │ 2026/06/01│ 詳細 │    │
│  │ タスクB   │ 作業中     │ 2026/06/02│ 詳細 │    │
│  │ タスクC   │ 完了       │ 2026/06/03│ 詳細 │    │
│  └─────────┴──────────┴──────────┴─────┘    │
│                                               │
└─────────────────────────────────────────────┘
```

### 表示項目

| 項目 | 表示内容 | 備考 |
|---|---|---|
| タイトル | タスクのタイトル | クリックで詳細画面へ遷移 |
| ステータス | TODO / IN_PROGRESS / DONE | 日本語表示（未着手 / 作業中 / 完了） |
| 作成日 | タスクの作成日時 | yyyy/MM/dd形式で表示 |
| 操作 | 「詳細」リンク | 詳細画面へ遷移 |

### 操作

| 操作 | 動作 | API |
|---|---|---|
| 「新規タスク作成」ボタン押下 | 新規作成ダイアログを表示 | — |
| ダイアログで「作成」ボタン押下 | タスクを作成し、一覧を再表示 | POST /api/tasks |
| 一覧行クリック / 「詳細」リンク押下 | タスク詳細画面へ遷移 | — |
| 画面表示時 | 全タスクを取得して一覧に表示 | GET /api/tasks |

### 新規タスク作成ダイアログ

| 入力項目 | 種別 | 必須 | バリデーション |
|---|---|---|---|
| タイトル | テキスト入力 | ○ | 1〜100文字 |
| 説明 | テキストエリア | × | 最大500文字 |

## 3. SCR-002 タスク詳細画面

### 画面レイアウト

```
┌─────────────────────────────────────────────┐
│  ← 一覧に戻る                                 │
├─────────────────────────────────────────────┤
│                                               │
│  タスク詳細                                    │
│                                               │
│  タイトル:  買い物リストを作る                    │
│  説明:     今週の食材を整理する                   │
│  ステータス: TODO                               │
│  作成日時:  2026/06/01 10:30                    │
│  更新日時:  2026/06/01 10:30                    │
│                                               │
│  ──────────────────────────                   │
│                                               │
│  ステータス変更:                                │
│  現在: TODO                                    │
│  次へ: [IN_PROGRESSに変更]                      │
│                                               │
│  ──────────────────────────                   │
│                                               │
│  [削除]                                        │
│                                               │
└─────────────────────────────────────────────┘
```

### 表示項目

| 項目 | 表示内容 | 備考 |
|---|---|---|
| タイトル | タスクのタイトル | |
| 説明 | タスクの説明文 | 未入力の場合は「（なし）」と表示 |
| ステータス | 現在のステータス | 日本語表示 |
| 作成日時 | 作成日時 | yyyy/MM/dd HH:mm形式 |
| 更新日時 | 最終更新日時 | yyyy/MM/dd HH:mm形式 |

### 操作

| 操作 | 動作 | API | 備考 |
|---|---|---|---|
| 「一覧に戻る」リンク押下 | タスク一覧画面へ遷移 | — | |
| 「IN_PROGRESSに変更」ボタン押下 | ステータスをIN_PROGRESSに更新 | PATCH /api/tasks/{id}/status | TODOの場合のみ表示 |
| 「DONEに変更」ボタン押下 | ステータスをDONEに更新 | PATCH /api/tasks/{id}/status | IN_PROGRESSの場合のみ表示 |
| 「削除」ボタン押下 | 確認ダイアログを表示 → 削除 → 一覧へ遷移 | DELETE /api/tasks/{id} | |

### ステータス変更ボタンの表示ルール

ステータスの遷移は一方向のみ。現在のステータスに応じて次に進めるボタンだけが表示される。

| 現在のステータス | 表示されるボタン |
|---|---|
| TODO | 「IN_PROGRESSに変更」 |
| IN_PROGRESS | 「DONEに変更」 |
| DONE | ボタンなし（変更不可） |

### 削除確認ダイアログ

- メッセージ: 「このタスクを削除しますか？この操作は取り消せません。」
- ボタン: 「削除する」「キャンセル」
